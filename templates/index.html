How it works (Character Autoencoder):
  • The model encodes each word into a 128-dimensional latent space
  • It creates a 10Ã—10 grid using parallelogram interpolation in latent space
  • Grid corners: [0,0]=Word1, [max,0]=Word2, [0,max]=Word3, [max,max]=Word2+Word3-Word1
  • Formula: point[i,j] = Word1 + (i/max)*(Word2-Word1) + (j/max)*(Word3-Word1)
  • Each point is decoded back to generate intermediate word forms
  • Only lowercase letters (a-z) supported, max 10 characters for autoencoder
